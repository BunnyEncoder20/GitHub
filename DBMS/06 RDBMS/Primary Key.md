- **Primary Key** is a key which uniquely identified each record in a table
- It is always **UNIQUE and NOT NULL** 

> Remember that keys are just attributes of a table 
> Candidate key is any attribute which has all unique records 
> Primary key is one selected from that group of candidate keys
